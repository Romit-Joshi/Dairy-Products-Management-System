# 🥛 Dairy Products Management System

A complete SQL-based Dairy Products Management System with an interactive Power BI dashboard for sales analysis and business insights.

---

## 📌 Project Overview

This project is designed to manage dairy products, customers, suppliers, orders, inventory, and payments using MySQL. It also includes a Power BI dashboard for visualizing sales performance.

---

## 🚀 Features

- Product Management
- Customer Management
- Supplier Management
- Inventory Management
- Order Management
- Payment Management
- Delivery Management
- Discount Management
- Stored Procedures
- Functions
- Views
- Triggers
- Sales Dashboard in Power BI
